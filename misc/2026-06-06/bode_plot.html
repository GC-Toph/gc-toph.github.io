阻尼、带宽与相位滞后：频域可视化

模型：标准二阶闭环 T(s)=ωn²/(s²+2ζωn s+ωn²)，再叠加纯延迟 e^{-sTd}。 拖动参数，观察幅频曲线、相频曲线和极点图如何同步变化。

参数

ζ 小：共振峰高、容易震荡。ζ 大：更稳但更慢。
ωn 越大，曲线整体向右移，系统愿意跟随更快变化。
延迟不改变幅值，但会额外增加相位滞后：-ωTd。
|T(jω)| = ωn² / √[(ωn²-ω²)² + (2ζωnω)²]
∠T(jω) = -atan2(2ζωnω, ωn²-ω²)
延迟相位 = -ωTd rad = -ωTd·180/π deg
带宽：|T(jωbw)| = 1/√2，即 -3 dB
怎么看这三张图？

幅频图看“系统会不会放大某些频率”。鼓包越高，共振越强，阻尼越不足。

带宽线看 -3 dB 交点。交点越靠右，系统能跟随越快的目标变化。

相频图看“输出比输入晚多少”。延迟只会把相位曲线向下拉，频率越高拉得越严重。

极点图看根的位置。极点越靠左衰减越快，越靠近虚轴越容易震荡。

幅频曲线：阻尼不足时会出现共振峰；-3 dB 交点定义闭环带宽。
相频曲线：橙色曲线包含纯延迟。注意延迟让高频相位快速下坠。
极点图：欠阻尼时是一对共轭复根；临界/过阻尼时变为实根。
