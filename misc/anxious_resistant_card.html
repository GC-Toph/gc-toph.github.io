焦虑型依恋｜关系触发急救卡

当晚回、沉默、冲突、距离变化让你突然失控时使用
先识别系统
核心判断：我此刻可能不是“太敏感”，而是依恋系统在扫描伴侣是否可得、是否回应、关系是否安全。
关系线索晚回 / 语气淡 / 计划变 / 冲突未修复
→
系统激活警觉升高 / 想靠近 / 难以停想
把三件事拆开
事实
可被录像记录的事：他说了什么、做了什么、什么时候发生。
解释
我脑内补出的含义：不爱我、要离开、我不够好。
需要
真正想要的是：确认、靠近、尊重、稳定回应。
30秒自检
身体：胸口紧、胃沉、心跳快、手冷、想盯手机。
冲动：追问、撤退、冷处理、翻旧账、提分手试探。
安全句：“我的感受是真的；但最坏解释只是一个假设。先多等一点信息，再行动。”
2

激活策略：脑内如何把你推向靠近

识别“想马上确认”的力量，不直接服从它
降激活
激活策略是推动你重新获得亲近和安全感的想法 / 感受。它的目标可以理解，但执行方式需要调整。
反复想
难以专注，脑中循环对方的语气、表情、已读未回、过去细节。
理想化
只记得对方好的部分，把“失去他/她”想成唯一灾难。
低估自己
觉得自己不够有吸引力、不够稳定、不值得被坚定选择。
唯一机会
相信“很难再遇到合适的人”，于是把不满足的关系合理化。
被激活时的执行流程
停

暂停即时反应：不发长消息、不下关系结论、不做惩罚性动作。

标

写一句：我被触发的是 ____；我担心的是 ____；我需要的是 ____。

查

至少列出2个中性解释：忙、累、误会、表达差异、需要时间整理。

说

等身体下降后，用“我感到 / 我需要 / 我希望”提出一个具体请求。

3

抗议行为：把“求安全”换成“可回应”

抗议行为会争取注意，但常常破坏真正的亲近
替代动作
旧动作
反复联系、冷处理、计分、敌意、威胁离开、制造嫉妒。
→
新动作
清楚表达不安，提出可执行请求，并保留对方回应空间。
想连发
改成一条清楚消息：“我有点不安。你方便时告诉我今晚大概几点联系？”
想冷掉
改成请求暂停：“我需要20分钟冷静，不是要消失，之后回来谈。”
想计分
改成说需求：“我不想用谁先低头衡量在乎，我想把问题讲清楚。”
想攻击
改成说脆弱：“我刚才生气下面其实是害怕被忽视。”
想分手试探
改成说边界：“我不想用威胁换回应，但这件事需要被认真处理。”
自问：我现在是在解决问题，还是在用惩罚、撤退、测试或讨好换取安全感？
4

有效沟通 + 安全型冲突原则

安全不是不冲突，而是冲突时仍能保护连接
照着说
有效沟通不是控诉对方，而是直接、具体、不过度包装地表达感受与需要。
“刚才发生的是：____。我脑中冒出的解释是：____，但我知道这不一定是事实。我的感受是：____。我需要 / 希望：____。你愿意和我一起确认一下吗？”
安全型冲突五原则
1关心对方福祉
不把关系变成输赢。
2聚焦当前问题
不翻旧账，不扩大战场。
3不人格化
说具体行为，不给对方定性。
4愿意留在对话里
可暂停，但不消失。
5表达需要
用请求替代审判。
✓修复句
“我被触发了，本意是靠近。”
顺序：先稳身体 → 再说事实 → 再说感受 → 再提请求 → 最后一起找下一步。
